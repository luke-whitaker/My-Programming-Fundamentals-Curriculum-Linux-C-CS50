# Linux, C, & CS50 Curriculum

I intend for this curriculum to consume between 5-10 hours per week for 14 weeks. I will be using my old 2010 MacBook Pro running **[Xubuntu 24.04](https://mirror.us.leaseweb.net/ubuntu-cdimage/xubuntu/releases/24.04.4/release/)** as my primary device/Operating System that I work from. Each week will include readings that lead feed into a lab, which will function as portfolio projects that I plan to publish here. 

## Phase 1: Foundations (weeks 1-3)

## Week 1: What Linux is and how it thinks
- [ ] Ch. 1 of How Linux Works (HLW)
- [ ] Ch. 1 of A Practical Guide to Linux (APGL)
- [ ] Intro + Ch. 1 of C Programming for Dummies (CPD)
- [ ] Week 0 of CS50
- [ ] LAB: Boot the Xubuntu machine. Run uname -r, free -h, lscpu. 

## Week 2: Basic Commands & The Shell - First Contact
- [ ] Ch. 2 HLW
- [ ] Ch. 2 & Ch. 3 APGL
- [ ] Ch. 2 & Ch. 3 CPD
- [ ] Week 1 of CS50
- [ ] LAB: Practice 20 core commands from memory. Write a C program that prints your name and today's date. Compile with gcc. 

## Week 3: Permissions, Processes & the Filesystem
- [ ] Ch. 2 Sections 2.15-2.21 HLW
- [ ] Ch. 4 APGL
- [ ] Ch. 4 CPD
- [ ] Week 2 CS50
- [ ] LAB: Create a directory structure, set file permissions, test sudo. Write a C program that does math and prints formatted results.
